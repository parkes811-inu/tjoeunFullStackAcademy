# axios 사용 방법
1. import
```js
import axios from 'axios';
```

2. GET
```js
axios.get("경로")
```

3. POST
```js
axios.post("경로", {데이터})
```

4. PUT
```js
axios.put("경로", {데이터})
```

5. DELETE
```js
axios.DELETE("경로")
```
